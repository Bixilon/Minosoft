# General GUI idea

## 2 Main GUI types

- Blocking/GUI (with mouse and keyboard interaction)
  - Inventories
  - Escape screen
  - Confirmation screen
  - Chat (input)
  - Command blocks, etc
- Non-blocking/HUD (~~permanent~~ overlay)
  - Chat messages
  - Scoreboard
  - Boss bar
  - Title
  - Hotbar
  - Debug info
  - Gadget text (like fps, time, etc)

## GUI components

- General
  - Text
  - Items
  - Container (list with items, the inventory and their positions)
- GUI
  - Buttons
  - TextField
  - Label
  - Checkbox
